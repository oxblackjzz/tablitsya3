## ?? Якщо щось пішло не так

### Railway не розгортає:

```bash
# Додайте railway.json в корінь проекту:
{
  "build": {
    "builder": "NIXPACKS"
  },
  "deploy": {
    "startCommand": "cd таблиця3 && dotnet run --urls http://0.0.0.0:$PORT",
    "restartPolicyType": "ON_FAILURE"
  }
}
```

### Azure показує помилку:

```bash
# Перевірте логи:
az webapp log tail --name tablitsya3-beta --resource-group tablitsya3-rg

# Перезапустіть:
az webapp restart --name tablitsya3-beta --resource-group tablitsya3-rg
```
