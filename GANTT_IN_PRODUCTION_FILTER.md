# ?? Фільтр "В роботі" на діаграмі Ганта

## ?? Що додано

Додано **новий режим фільтрації** безпосередньо в компоненті `GanttChart` - кнопка **"В роботі"**.

---

## ? Нова можливість

### Панель фільтрів на діаграмі:

```
?????????????????????????????????????????????????
?  Режим показу:    ?
?  [? Незавершені] [?? В роботі] [?? Усі]      ?
?????????????????????????????????????????????????
```

### Три режими перегляду:

1. **? Незавершені** (жовта кнопка)
   - Показує замовлення, які ще не завершено
   - Виключає замовлення з датою завершення < сьогодні

2. **?? В роботі** (зелена кнопка) - **НОВИЙ!**
   - Показує тільки замовлення, що зараз виробляються
   - Перевіряє чи сьогоднішня дата в діапазоні виробництва

3. **?? Усі** (сіра кнопка)
   - Показує всі замовлення без фільтрації

## ?? Логіка фільтрації

### Метод IsOrderInProgress()

```csharp
protected bool IsOrderInProgress(Order order)
{
    var position = cachedOrderPositions.FirstOrDefault(p => p.OrderDay == order.Day);
    
    if (position == null)
    {
        // Fallback: якщо немає позиції
        if (order.CompletionDate < DateTime.Today)
        {
   return false; // Вже завершено
        }
        
        var daysUntilCompletion = (order.CompletionDate - DateTime.Today).Days;
        return order.StartDate <= DateTime.Today && 
 daysUntilCompletion >= 0 && 
    daysUntilCompletion <= 7;
    }
    
    var today = DateTime.Today;
    bool productionStarted = today >= position.ProductionStartDate.Date;
    bool productionNotCompleted = today <= position.ProductionEndDate.Date;

    return productionStarted && productionNotCompleted;
}
```

### Алгоритм перевірки:

```
1. Знайти позицію замовлення (ProductionStartDate, ProductionEndDate)
2. Перевірити: 
   ? Сьогодні >= Дата початку виробництва
   ? Сьогодні <= Дата закінчення виробництва
3. Якщо обидві умови TRUE ? замовлення "В роботі"
```

### Загальний фільтр (ProductionPlanning):
```
? Коли потрібно побачити однаковий стан на всіх цехах
```

```
? Коли потрібно побачити різні стани на різних цехах
? Коли потрібна швидка перевірка одного цеху
? Коли не потрібна таблиця (тільки діаграма)
```

## ?? Можливі питання

### Q: Чому замовлення не показується в "В роботі"?

**A:** Перевірте дати:
```
Замовлення №5:
• Дата замовлення: 25.10.2025
• Початок виробництва: 31.10.2025
• Закінчення виробництва: 08.11.2025
• Сьогодні: 03.11.2025

? 03.11 між 31.10 і 08.11 ? показується
```

### Q: Чому показується багато замовлень?

**A:** Можливі причини:
1. Великий цех з високою потужністю
2. Довгі терміни виробництва (виробляється паралельно)
3. Перекриття дат виробництва

### Q: Як побачити тільки одне замовлення?

**A:** Використайте фільтр "Період":
```
1. Встановити "Від" і "До" на потрібні дати
2. Натиснути "Застосувати"
```
