### Де зберігається дата завершення?

```csharp
// Цех №3, Замовлення №5 -> "3_5": "2025-01-15"
```

### Як використовується при перерахунку?

У `ProductionPlanningService.CalculateSchedule()`:

```csharp
// Перевірка чи є кастомна дата завершення
var customKey = $"{workshopNumber}_{day}";
if (customCompletionDates?.ContainsKey(customKey) == true)
{
    // Використовуємо користувацьку дату
    order.CompletionDate = customCompletionDates[customKey];
}
else
{
    // Розраховуємо автоматично
    order.CompletionDate = order.ProductionStartDate.AddDays(order.ProductionDuration);
}
```
