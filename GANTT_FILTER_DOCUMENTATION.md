## ?? Технічна реалізація

### ProductionPlanning.razor.cs

```csharp
// Фільтр замовлень (для таблиць)
private string orderFilter = "all";
private DateTime filterDate = DateTime.Today;

// ?? Окремий фільтр для діаграм Ганта
private string ganttFilter = "all";
private DateTime ganttFilterDate = DateTime.Today;
```

### Методи фільтрації

#### 1. FilterOrders - для таблиць
```csharp
private IEnumerable<Order> FilterOrders(IEnumerable<Order> orders)
{
    return orderFilter switch
    {
        "in-production" => orders.Where(o => o.IsInProduction(filterDate)),
        "not-started" => orders.Where(o => o.IsNotStarted(filterDate)),
   "completed" => orders.Where(o => o.IsCompleted(filterDate)),
        _ => orders.OrderBy(o => o.StartDate)
    };
}
```

#### 2. FilterGanttOrders - для діаграм
```csharp
private IEnumerable<Order> FilterGanttOrders(IEnumerable<Order> orders)
{
    return ganttFilter switch
    {
        "in-production" => orders.Where(o => o.IsInProduction(ganttFilterDate)),
        _ => orders.OrderBy(o => o.StartDate)
    };
}
```

## ?? Сценарії використання

### Сценарій 1: Перегляд поточної роботи

**Користувач:** Хочу побачити що зараз виробляється

**Дії:**
1. Відкрити сторінку "Планування виробництва"
2. В панелі "Фільтр діаграм" вибрати: **"Тільки в роботі"**

**Діаграми:**

- Дата: 03.11.2025

**Таблиці:**

- Дата: 03.11.2025

**Результат:**
- Діаграми показують тільки замовлення в роботі
- Таблиці показують всі замовлення з позначкою статусу

## ?? Переваги

### 1. **Гнучкість**
? Можна дивитися різні дані на діаграмах і в таблицях одночасно

### 2. **Зручність**
? Швидкий перегляд поточного завантаження цехів

### 3. **Не заважає**
? Фільтри працюють незалежно - зміна одного не впливає на інший

### 4. **Візуалізація**
? На діаграмі відразу видно кількість замовлень у відфільтрованому режимі

2. **За періодом**
   ```
   [Сьогодні] [Цей тиждень] [Цей місяць] [Власний]
   ```

3. **За замовниками**
   ```
   [Замовник А] [Замовник Б] [Усі]
   ```

4. **За обсягом**
   ```
 [Малі (<1000 м?)] [Середні] [Великі (>5000 м?)]
   ```

## ?? Відомі обмеження

1. **Фільтр діаграм**
   - Тільки 2 опції: "Усі" і "Тільки в роботі"
   - Немає фільтрів "Очікують" і "Завершено"

2. **Синхронізація дат**
   - Дати фільтрів не синхронізуються автоматично
   - Користувач повинен вручну встановлювати обидві дати
