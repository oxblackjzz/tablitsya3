## ? Часті питання:

### Q: Чи потрібен мені окремий сервер?

**A: НІ!** Ваш Blazor Server проект - це вже веб-сервер. Ви просто публікуєте його на хостинг.

### Q: Чи потрібна база даних?

**A: НІ для бета!** Дані зберігаються в localStorage браузера. Для production додасте пізніше (Azure SQL / PostgreSQL).

### Q: Скільки це коштує?

**A: $0 для бета!** Render безкоштовний (засинає після 15 хв), Railway $5/міс (не засинає).

### Q: Скільки часу?

**A: 10 хвилин!** 3 команди + 5 кліків мишкою.

### Q: Що якщо щось зламається?

**A: Все відновлюється!** Render/Railway автоматично перезапускаються. Логи доступні в dashboard.

### Q: Як оновити код?

**A: Просто git push!**
```powershell
git add .
git commit -m "Update"
git push
```
Render/Railway автоматично перезапустять сайт.
