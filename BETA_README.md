## ?? Про систему

Веб-додаток для планування виробництва з візуалізацією завантаження цехів через діаграми Ганта.

### ? Основні функції:

- ?? **Діаграми Ганта** для 3 цехів (№1, №3, №6)
- ?? **Календарне планування** з урахуванням робочих/вихідних днів
- ?? **Розрахунок завантаження** цехів в реальному часі
- ?? **Управління замовленнями** (додавання, редагування, видалення)
- ?? **Фільтри** за статусом та періодом
- ?? **Збереження даних** локально в браузері

## ?? Швидкий старт

### 1?? Додавання замовлень

1. Натисніть **"Додати замовлення"** в меню
2. Оберіть цех (№1, №3, або №6)
3. Заповніть дані:
   - Назва замовлення
   - Площа (м?)
   - Дата замовлення
4. Натисніть **"Додати замовлення"**
5. Після додавання всіх замовлень ? **"Повернутися до планування"**

### 2?? Розрахунок графіків

1. На сторінці "Планування виробництва"
2. Натисніть **"Розрахувати графіки"**
3. Система автоматично:
   - Розподілить замовлення по часу
   - Врахує потужність цехів
   - Визначить дати виробництва

### 3?? Перегляд діаграм

**Фільтри:**
- **? Незавершені** - замовлення що очікують або в роботі
- **?? В роботі** - замовлення що виробляються зараз
- **?? Усі** - всі замовлення

**Користувацький період:**
- Виберіть дати "Від" та "До"
- Натисніть **"Застосувати"**

**Деталі замовлення:**
- Наведіть курсор на блок замовлення
- З'явиться підказка з деталями

### 4?? Редагування замовлень

1. Прокрутіть до таблиці замовлень
2. Натисніть кнопку **"Редагувати"** (олівець)
3. Змініть дані
4. Натисніть **"Зберегти"** (галочка)
5. Графіки автоматично оновляться

## ?? Безпека та конфіденційність

### Де зберігаються дані?

**Локально в браузері:**
- Всі дані зберігаються в `localStorage`
- Нічого не передається на сервер
- Очищення даних браузера = видалення всіх замовлень

### Як зберегти дані?

**Резервне копіювання:**
1. Відкрийте консоль браузера (F12)
2. Вкладка "Application" ? "Local Storage"
3. Знайдіть ключ `workshop-data`
4. Скопіюйте значення в текстовий файл

**Відновлення:**
1. Відкрийте консоль браузера (F12)
2. Виконайте:
```javascript
localStorage.setItem('workshop-data', 'ваші_збережені_дані');
location.reload();
```

## ?? Часті питання (FAQ)

### Q: Чому сайт повільно завантажується?

**A:** Безкоштовний сервер "засинає" після неактивності. Перше завантаження після сну займає 10-30 секунд. Наступні завантаження будуть швидкими.

### Q: Чому мої дані зникли?

**A:** Дані зберігаються локально в браузері. Можливі причини:
- Очистили кеш браузера
- Використали інший браузер/комп'ютер
- Режим інкогніто

**Рішення:** Робіть резервні копії (див. розділ "Безпека").

### Q: Чому замовлення накладаються?

**A:** Це нормально! Система показує що цех перевантажений. Рішення:
- Збільште потужність цеху
- Перенесіть замовлення на інший цех
- Змініть дати замовлень

### Q: Як видалити всі дані?

**A:** Натисніть "Очистити все" на сторінці планування.

### Q: Чи можна використовувати на телефоні?

**A:** Частково. Деякі функції можуть працювати незручно. Рекомендуємо комп'ютер або планшет.
