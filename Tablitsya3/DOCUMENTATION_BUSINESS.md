# 🏭 Tablitsya3 — Система управління меблевим виробництвом

## Короткий опис для керівництва

---

## 📌 Що це за система?

**Tablitsya3** — це програмне забезпечення, яке допомагає меблевому підприємству:

✅ **Контролювати виробництво** — бачити де знаходиться кожна деталь меблів  
✅ **Планувати роботу** — автоматично розраховувати терміни виконання замовлень  
✅ **Відстежувати працівників** — хто що зробив і скільки  
✅ **Зменшувати брак** — фіксувати дефекти та контролювати якість  
✅ **Економити час** — автоматизувати рутинні процеси

## 🏭 Етапи виробництва

Кожна деталь меблів проходить **5 етапів**. Система відстежує кожен з них:

```
    ┌─────────┐      ┌─────────┐      ┌─────────┐      ┌─────────┐      ┌─────────┐
    │    1    │      │    2    │      │    3    │      │    4    │      │    5    │
    │         │      │         │      │         │      │         │      │         │
    │ 🪚      │ ───▶ │ 📐      │ ───▶ │ 🔩      │ ───▶ │ 📦      │ ───▶ │ 📮      │
    │         │      │         │      │         │      │         │      │         │
    │ ПОРІЗКА │      │ ПОКЛЕЙКА│      │СВЕРДЛІННЯ│     │СОРТУВАННЯ│     │ПАКУВАННЯ│
    │         │      │ КРОМКИ  │      │         │      │         │      │         │
    └─────────┘      └─────────┘      └─────────┘      └─────────┘      └─────────┘
    
    Розкрій         Наклеювання      Отвори для       Групування      Готово до
    листів          декоративної     фурнітури        по              відправки
    ДСП/МДФ         стрічки          та з'єднань      замовленнях     клієнту
```

### Що відбувається на кожному етапі:

| Етап | Що робиться | Що фіксує система |
|------|-------------|-------------------|
| **1. Порізка** | Великий лист ДСП ріжуть на деталі потрібного розміру | ✅ Деталь вирізана, час, працівник |
| **2. Поклейка** | На торці деталі клеять декоративну кромку | ✅ Кромка наклеєна (може бути 1-4 сторони) |
| **3. Свердління** | Роблять отвори для петель, ручок, з'єднань | ✅ Отвори готові, час, працівник |
| **4. Сортування** | Деталі групують по замовленнях | ✅ Деталь в правильній групі |
| **5. Пакування** | Загортають та готують до відправки | ✅ Замовлення готове до відвантаження |

## 🤖 Як працює автоматичне сканування?

### Принцип роботи:

Сканування відбувається **автоматично** за допомогою **промислових сканерів**, встановлених на кожній робочій станції. Працівник просто виконує свою роботу — система сама фіксує проходження деталей.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                      АВТОМАТИЧНЕ СКАНУВАННЯ НА ВИРОБНИЦТВІ                       │
└─────────────────────────────────────────────────────────────────────────────────┘

                    🏭 РОБОЧА СТАНЦІЯ (наприклад, розкрійний центр)
      
      ┌─────────────────────────────────────────────────────────────────┐
      │                                                                 │
      │   ┌─────────┐      ┌─────────────────┐      ┌─────────┐        │
      │   │         │      │                 │      │         │        │
      │   │ 📥 ВХІД │ ───▶ │ ⚙️ ОБРОБКА      │ ───▶ │ 📤 ВИХІД│        │
      │   │ деталей │      │ (працівник      │      │ деталей │        │
      │   │         │      │  працює)        │      │         │        │
      │   └────┬────┘      └─────────────────┘      └────┬────┘        │
      │        │                                        │              │
      │        ▼                                        ▼              │
      │   ┌─────────┐                              ┌─────────┐         │
      │   │ 📷      │                              │ 📷      │         │
      │   │ СКАНЕР  │                              │ СКАНЕР  │         │
      │   │ (авто)  │                              │ (авто)  │         │
      │   └────┬────┘                              └────┬────┘         │
      │        │                                        │              │
      └────────│────────────────────────────────────────│──────────────┘
               │                                        │
               ▼                                        ▼
         ┌──────────┐                            ┌──────────┐
         │ 📡 Деталь│                            │ 📡 Деталь│
         │ прийнята │                            │ оброблена│
         │ в роботу │                            │ ✅       │
         └──────────┘                            └──────────┘
               │                                        │
               └──────────────────┬─────────────────────┘
                                  ▼
                           ┌────────────┐
                           │ 💾 СИСТЕМА │
                           │ автоматично│

                           └────────────┘
```

### Переваги автоматичного сканування:

| Перевага | Опис |
|----------|------|
| ⚡ **Швидкість** | Сканування за 0.1 секунди, не затримує виробництво |
| 🎯 **Точність** | Жодних помилок людського фактору |
| 👷 **Зручність** | Працівник не відволікається від роботи |
| 📊 **Повнота** | 100% деталей фіксуються автоматично |
