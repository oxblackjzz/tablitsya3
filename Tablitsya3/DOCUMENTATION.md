## 1. Огляд системи

**Tablitsya3** — це комплексна система управління меблевим виробництвом, побудована на ASP.NET Core Blazor Server. Система призначена для:

- 📦 **Управління замовленнями** — планування та розподіл замовлень по цехах
- 🏭 **Виробничий контроль** — відстеження деталей на кожному етапі виробництва
- 👷 **Управління персоналом** — авторизація працівників, KPI, облік робочого часу
- 📊 **Аналітика** — статистика, звітність, моніторинг продуктивності
- 🔍 **QR-сканування** — відстеження деталей через QR-коди

### Технологічний стек

| Компонент | Технологія |
|-----------|------------|
| Frontend | Blazor Server |
| Backend | ASP.NET Core 8.0 |
| ORM | Entity Framework Core |
| Database | PostgreSQL / SQLite |
| Containerization | Docker |

### 7.2 Part (модель деталі)
```csharp
public class Part
{
    // Ідентифікація
    public int Id { get; set; }
    public string ProjectExternalUuid { get; set; }
    public int PartId { get; set; }
    public int PartCounter { get; set; }
    
    // Властивості
    public string Name { get; set; }
    public string Code { get; set; }
    public double Length { get; set; }   // мм
    public double Width { get; set; }    // мм
    public double Thickness { get; set; } // мм
    public string Material { get; set; }
    
    // Статуси етапів
    public bool IsCutCompleted { get; set; }
    public bool IsEdgeBandingCompleted { get; set; }
    public bool IsDrillingCompleted { get; set; }
    public bool IsSortingCompleted { get; set; }
    public bool IsPackingCompleted { get; set; }
    
    // Вимоги до етапів
    public bool RequiresCutting { get; set; } = true;
    public bool RequiresEdgeBanding { get; set; } = true;
    public bool RequiresDrilling { get; set; } = true;
    public bool RequiresSorting { get; set; } = true;
    public bool RequiresPacking { get; set; } = true;
    
    // Методи
    public ProductionStage? CurrentStage { get; }
    public bool IsStageRequired(ProductionStage stage);
    public bool IsStageCompleted(ProductionStage stage);
    public bool CanAdvanceToStage(ProductionStage stage);
    public static Part ParseQRCode(string qrCode);
    public string GenerateQRCode();
}
```
